# Play Store Listing Draft

## App Name
AetherQuest: Legends of the Fallen Sky

## Short Description
Story-driven fantasy RPG with branching endings, quests, and tactical battles.

## Full Description
AetherQuest is a Kotlin-built RPG featuring chapter-based storytelling, world-region progression, and meaningful choices.

Key features:
- Three hero classes with distinct playstyles
- Turn-based combat with skills, loot, and upgrades
- Branching narrative with romance/loyalty routes
- Multiple endings based on your choices
- Offline local save progression

## Keywords
